# Simple Calculator Using C

## Project Overview
This is a simple calculator developed using C programming language. The calculator performs basic arithmetic operations such as addition, subtraction, multiplication, and division.

## Technologies Used
- C Programming
- Online C Compiler

## How to Run
1. Compile the code using a C compiler.
2. Run the executable file.
3. Enter two numbers and select an operation.
4. View the result.
